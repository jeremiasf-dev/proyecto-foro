# proyecto-foro

Un espacio de intercambio a ritmo humano

Principios:

Ritmo humano
El sistema respeta el tiempo de las personas. No empuja a reaccionar ni a producir con urgencia.

Intercambio
La participación implica ida y vuelta. No es exhibición de contenido ni consumo pasivo (en lo posible...)

Espacio
No es un feed. Es un lugar donde quedarse, leer, pensar y volver.

Lo que el foro NO es:

No es un feed infinito
El contenido tiene cierre. No incentiva el desplazamiento sin fin.

No es un sistema de recompensas
No hay puntos, likes dopaminérgicos, rachas ni monetización de la atención.

No es un espacio de urgencia
Responder tarde es válido. El silencio no es un error.

No es un espacio de descarga emocional
El conflicto puede existir, pero debe estar argumentado. El forobardo queda afuera.

No es un sitio de noticias
No se publican noticias, primicias ni contenido atado a la coyuntura inmediata.
El proyecto no compite por actualidad ni por atención instantánea.

Lo que el foro SÍ incentiva

Respeto
Principio no negociable.

Buena fe
Se escribe para entender y ser entendido, no para descargar problemas en forma de insultos y bronca.

Claridad
Las ideas simples son bienvenidas. No se exige profundidad artificial ni extensión forzada.

##########################################################################################################

Esto puede expandirse, pero los principios antes mencionados son innegociables.
